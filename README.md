# PhaDataPro AI — Чат-ассистент

Русскоязычный AI-ассистент для аналитической платформы **PhaDataPro** (Viortis).  
Помогает пользователям строить отчёты, анализировать данные, отвечает на вопросы по всем модулям и каналам системы.

**Стек:** Streamlit · OpenAI GPT-4o · RAG (text-embedding-3-small) · Python 3.11+

## Быстрый старт (локально)

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Создайте файл секретов

```bash
cp .streamlit/secrets.toml.example .streamlit/secrets.toml
```

Откройте `.streamlit/secrets.toml` и заполните:

```toml
OPENAI_API_KEY = "sk-proj-ВАШ_КЛЮЧ"

[auth]
admin_login    = "admin"
admin_password = "ВАШ_НАДЁЖНЫЙ_ПАРОЛЬ"

user1_login    = "viortis"
user1_password = "ПАРОЛЬ_КЛИЕНТА"
```

> ⚠️ `.streamlit/secrets.toml` добавлен в `.gitignore` — он **никогда не попадёт в Git**.

### 3. Запустите

```bash
streamlit run streamlit_app.py
```

Откроется браузер: http://localhost:8501

## Деплой на Streamlit Community Cloud

### Требования
- Репозиторий на GitHub (публичный или приватный)
- Аккаунт на [streamlit.io](https://streamlit.io)

### Шаги деплоя

**1. Проверьте, что секреты не попали в репо:**
```bash
git status
git diff --cached
```

**2. Запушьте код на GitHub:**
```bash
git push origin main
```

**3. Создайте приложение на Streamlit Cloud:**
- Зайдите на [share.streamlit.io](https://share.streamlit.io)
- New app → выберите репозиторий
- Main file path: `streamlit_app.py`
- Нажмите Deploy

**4. Добавьте Secrets в Streamlit Cloud:**
- Перейдите: App settings → Secrets
- Вставьте содержимое (без реальных значений — введите свои):

```toml
OPENAI_API_KEY = "sk-proj-ВАШ_КЛЮЧ"

[auth]
admin_login    = "admin"
admin_password = "ВАШ_НАДЁЖНЫЙ_ПАРОЛЬ"

user1_login    = "viortis"
user1_password = "ПАРОЛЬ_КЛИЕНТА"
```

- Нажмите Save → приложение перезапустится автоматически

### Обновление (повторный деплой)
```bash
git add .
git commit -m "fix: описание изменений"
git push origin main
```
Streamlit Cloud подхватит изменения автоматически через ~1 минуту.

## Добавить нового клиента

В Streamlit Cloud Secrets добавьте строки:
```toml

# ... существующие ...
user2_login    = "новый_логин"
user2_password = "НАДЁЖНЫЙ_ПАРОЛЬ"
```
Сохраните — приложение перезапустится, новый пользователь сразу получит доступ.
