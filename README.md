# PhaDataPro AI — Чат-ассистент

Русскоязычный AI-ассистент для аналитической платформы **PhaDataPro** (Viortis).  
Помогает пользователям строить отчёты, анализировать данные, отвечает на вопросы по всем модулям и каналам системы.

**Стек:** Streamlit · OpenAI GPT-4o · RAG (text-embedding-3-small) · Python 3.11+

#### ➕ Добавленная информация (в `modules_guide.md`)

**Механика взаимодействия:**
- Как добавить фильтр: нажать значение → попадает в блок **«Выбрано»**
- Как выбрать блок: нажать → выделится **красным**, ещё раз → снять
- Как сбросить все блоки: кнопка **«Отжать все кубы»**

**Условия построения отчёта:**
- Минимум: **1 блок наименований** + **1 блок числовых показателей**
- Период: выбрать интервал + нажать **«+»** для добавления (можно неограниченное кол-во)

**Расчёты и ограничения:**
- **Прирост** рассчитывается **только при двух и более периодах** (одного периода недостаточно)
- **Объединение периодов** — суммирует данные за выбранные периоды
- **Сохранение отчётов** — максимум **4 одновременно** (галочка «Сохранить отчет» перед скачиванием)
- **Рекомендация:** если отчёт > 500 строк → скачивать в Excel вместо онлайн-просмотра

#### 🐛 Баги в коде (исправлены в `streamlit_app.py`)

1. **Уточняющие ответы не логировались**
   - Проблема: когда бот просил уточнить модуль/канал, ответ не попадал в логи админ-панели
   - Решение: все уточнения теперь проходят через функцию `_finish()` с логированием

2. **Ошибки записи логов глушились**
   - Проблема: если запись лога не удавалась, ошибка исчезала молча (`except Exception: pass`)
   - Решение: ошибки теперь отображаются пользователю через `st.warning()`

3. **Админ-панель не показывала уточнения**
   - Проблема: в фильтре Intent не было типа `clarification`
   - Решение: добавлен `clarification` в список доступных intent для фильтрации

## Быстрый старт (локально)

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Создайте файл секретов

```bash
cp .streamlit/secrets.toml.example .streamlit/secrets.toml
```

Откройте `.streamlit/secrets.toml` и заполните:

```toml
OPENAI_API_KEY = "sk-proj-ВАШ_КЛЮЧ"

[auth]
admin_login    = "admin"
admin_password = "ВАШ_НАДЁЖНЫЙ_ПАРОЛЬ"

user1_login    = "viortis"
user1_password = "ПАРОЛЬ_КЛИЕНТА"
```

> ⚠️ `.streamlit/secrets.toml` добавлен в `.gitignore` — он **никогда не попадёт в Git**.

### 3. Запустите

```bash
streamlit run streamlit_app.py
```

Откроется браузер: http://localhost:8501
